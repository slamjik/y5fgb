# Локальная desktop-папка (игнорируется Git)

Эта папка нужна, чтобы не смешивать web-основной поток разработки с локальными desktop-экспериментами.

## Как использовать

- Кладите сюда любые временные desktop-файлы и черновики.
- Эти файлы **не попадут в git** (кроме этого README).
- Основной desktop-код проекта остаётся в `apps/client-desktop` и продолжает работать как раньше.

## Зачем это нужно

- Web остаётся основной целью в репозитории.
- Desktop-локальные правки не мешают web-коммитам.
- Без поломки текущей архитектуры и сборочных скриптов.
