# User Guide (RU / EN)

## Install
- Windows: install generated NSIS package from `src-tauri/target/release/bundle/nsis`.
- Linux: use AppImage or DEB from `src-tauri/target/release/bundle`.

## First Run / Первый запуск
- EN: Sign in or register, then complete onboarding checklist.
- RU: Войдите или зарегистрируйтесь, затем пройдите onboarding-чеклист.

## Add Device / Добавление устройства
- EN: Login from new desktop, device appears as `pending`.
- RU: Войдите с нового desktop, устройство появится как `pending`.
- EN: Approve from trusted device on `Devices` screen.
- RU: Подтвердите его с trusted-устройства на экране `Devices`.

## Recovery / Восстановление
- EN: Use recovery flow with offline recovery code.
- RU: Используйте recovery flow и офлайн recovery code.

## Messaging Basics / Основы messaging
- EN: Create direct/group conversation, send messages, retry from Outbox if needed.
- RU: Создайте личный/групповой чат, отправляйте сообщения, при ошибках используйте retry в Outbox.

## Plugins / Плагины
- EN: Open `Plugins`, discover/install/enable, review permissions before enabling.
- RU: Откройте `Plugins`, выполните discover/install/enable и проверяйте разрешения перед включением.
